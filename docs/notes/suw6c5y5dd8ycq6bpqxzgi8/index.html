身份识别与身份验证

概念

身份识别是声称某人或某物是什么，而身份验证则确定该声明是否属实。

身份识别

  • 声称自己是谁
  • 身份证实
  • 伪造身份

身份验证

  • 因子
  • 多因子身份验证
  • 双向验证（中间人攻击）
